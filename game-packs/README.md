# Игровые пакеты (Game Packs)

Эта папка содержит JSON файлы с вопросами и ответами для игры "Сто к Одному".

## Формат пакета

Каждый пакет должен соответствовать следующей структуре:

```json
{
  "pack_id": "unique_id",
  "title": "Название пакета",
  "description": "Описание пакета",
  "rounds": {
    "simple": {
      "question": "Вопрос для простого раунда?",
      "answers": [
        {"text": "Ответ 1", "score": 30},
        {"text": "Ответ 2", "score": 25},
        {"text": "Ответ 3", "score": 15},
        {"text": "Ответ 4", "score": 10},
        {"text": "Ответ 5", "score": 12},
        {"text": "Ответ 6", "score": 8}
      ]
    },
    "double": {
      "question": "Вопрос для двойного раунда?",
      "answers": [
        // То же самое 6 ответов
      ]
    },
    "triple": {
      "question": "Вопрос для тройного раунда?",
      "answers": [
        // То же самое 6 ответов
      ]
    },
    "reverse": {
      "question": "Вопрос для игры наоборот?",
      "answers": [
        // То же самое 6 ответов
      ]
    },
    "big_game": [
      {
        "question": "Вопрос для большой игры 1?",
        "answers": [
          {"text": "Ответ", "score": 25, "synonyms": ["Синоним1", "Синоним2"]},
          // Минимум 3 ответа, но можно больше
        ]
      },
      // Всего 5 вопросов для большой игры
    ]
  }
}
```

## Правила

### Общие требования

1. **pack_id**: Уникальный идентификатор пакета (латинские буквы, цифры, подчеркивания)
2. **title**: Понятное название пакета
3. **description**: Краткое описание тематики пакета
4. Обязательно наличие всех 5 раундов: simple, double, triple, reverse, big_game

### Раунды

#### Simple (Простой раунд)
- Один вопрос
- Точно 6 ответов
- Очки от 5 до 50
- Самый популярный ответ - больше всего очков

#### Double (Двойной раунд)
- Один вопрос
- Точно 6 ответов
- Очки от 10 до 100
- Ведущий удваивает очки при начислении

#### Triple (Тройной раунд)
- Один вопрос
- Точно 6 ответов
- Очки от 15 до 150
- Ведущий утраивает очки при начислении

#### Reverse (Игра наоборот)
- Один вопрос
- Точно 6 ответов
- Очки от 5 до 50
- Самый редкий ответ (низкий в опросе) - больше всего очков

#### Big Game (Большая игра)
- Ровно 5 вопросов
- Минимум 3 ответа на каждый вопрос, максимум не ограничен
- Поле `synonyms` (опционально): массив синонимов для ответа

### Очки

Рекомендуемые диапазоны:
- Simple: 5-50 очков
- Double: 10-100 очков  
- Triple: 15-150 очков
- Reverse: 5-50 очков
- Big Game: 10-50 очков

Сумма очков всех ответов в вопросе обычно составляет около 100 очков (для simple/double/triple/reverse).

### Темы вопросов

Хорошими темами являются:
- Бытовые ситуации
- Популярная культура
- Еда и напитки
- Транспорт
- Работа и учеба
- Семья и отношения
- Праздники и традиции
- Природа и животные
- Технологии
- Спорт

Избегайте:
- Контентов для взрослых
- Политики и религии
- Оскорблений и дискриминации
- Слишком узких или специфических тем

## Создание нового пакета

1. Создайте новый файл `.json` в этой папке
2. Скопируйте структуру из `pack1.json`
3. Заполните своими вопросами
4. Проверьте что формат соответствует требованиям
5. Добавьте файл `pack_id` в выпадающий список в `client/src/App.jsx`
6. Протестируйте пакет в игре

## Валидация

Используйте этот JSON Schema для валидации:

```json
{
  "type": "object",
  "required": ["pack_id", "title", "rounds"],
  "properties": {
    "pack_id": {"type": "string"},
    "title": {"type": "string"},
    "description": {"type": "string"},
    "rounds": {
      "type": "object",
      "required": ["simple", "double", "triple", "reverse", "big_game"],
      "properties": {
        "simple": {
          "type": "object",
          "required": ["question", "answers"],
          "properties": {
            "question": {"type": "string"},
            "answers": {
              "type": "array",
              "items": {
                "type": "object",
                "required": ["text", "score"],
                "properties": {
                  "text": {"type": "string"},
                  "score": {"type": "number"}
                }
              },
              "minItems": 6,
              "maxItems": 6
            }
          }
        },
        "big_game": {
          "type": "array",
          "items": {
            "type": "object",
            "required": ["question", "answers"],
            "properties": {
              "question": {"type": "string"},
              "answers": {
                "type": "array",
                "items": {
                  "type": "object",
                  "required": ["text", "score"],
                  "properties": {
                    "text": {"type": "string"},
                    "score": {"type": "number"},
                    "synonyms": {"type": "array", "items": {"type": "string"}}
                  }
                }
              }
            }
          },
          "minItems": 5,
          "maxItems": 5
        }
      }
    }
  }
}
```

## Поддерживаемые языки

Пакеты могут быть на любом языке, но убедитесь что:
- Все ответы одной вопроса на одном языке
- Документация к пакету на том же языке
