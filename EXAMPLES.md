# Примеры использования и демо-сценарии

Этот документ содержит примеры использования приложения "Сто к Одному" для различных ситуаций.

## Сценарий 1: Вечеринка с друзьями

### Настройка
1. Заходите на `http://localhost:3000`
2. Нажимаете "Создать игру"
3. Выбираете "Пакет для дня рождения"
4. Получаете код комнаты (например, "XK42")

### Игровые устройства
- **Табло**: Ноутбук подключенный к телевизору по HDMI
  - URL: `http://localhost:3000/game/XK42/board`
  - Открываете на большом экране, чтобы все видели вопросы и ответы
  
- **Ведущий**: Планшет или второй ноутбук
  - URL: `http://localhost:3000/game/XK42/admin`
  - Вы управляете игрой, открываете ответы, начисляете очки
  
- **Кнопки игроков**: 2 смартфона
  - URL: `http://localhost:3000/game/XK42/buzzer`
  - Игроки выбирают команду (1 или 2)
  - При нажатии кнопки определяется кто первым ответил

### Игровой процесс
1. Выбираете "Простой раунд"
2. На табло появляется вопрос "Самый популярный фрукт?"
3. Игроки нажимают кнопки
4. Побеждает тот, кто нажал первым (показывается на табло)
5. Игрок говорит: "Яблоко"
6. Вы (ведущий) смотрите в свой список ответов
7. Находите "Яблоко" (30 очков), нажимаете "Открыть"
8. На табло плашка переворачивается, играет звук "Дзинь"
9. Очки добавляются в банк
10. Следующий ответ: "Банан" → открываете
11. Если ответа нет → нажимаете "Промах"
12. После 3 промахов → нажимаете "Передача хода"
13. В конце раунда → нажимаете "Начислить очки"

## Сценарий 2: Корпоративная вечеринка

### Специфика
- Используйте "Корпоративная вечеринка" пакет
- Вопросы адаптированы для офисной тематики
- Можно создать свой пакет с вопросами про вашу компанию

### Советы
- Подготовьте список участников beforehand
- Разделите на команды по отделам
- Используйте проектор для табло
- Подготовьте звуковые колонки для звуков

## Сценарий 3: День рождения детей

### Настройка
- Создайте свой пакет с детскими вопросами
- Упростите очки (все ответы по 10 очков)
- Используйте только простые раунды

### Пример детского пакета
```json
{
  "pack_id": "kids",
  "title": "Детский день рождения",
  "rounds": {
    "simple": {
      "question": "Любимая игрушка?",
      "answers": [
        {"text": "Кукла", "score": 10},

        {"text": "Мяч", "score": 10},
        {"text": "Конструктор", "score": 10},
        {"text": "Плюшевая игрушка", "score": 10},
        {"text": "Кубики", "score": 10}
      ]
    }
  }

```

## Расширенные сценарии

### Создание турнира

1. Подготовьте несколько пакетов
2. Создайте несколько комнат
3. Проводите параллельные игры
4. Ведущий наблюдает за несколькими комнатами (открывает несколько вкладок admin)

### Использование с большим количеством участников

1. Создайте несколько кнопок на каждую команду
2. Игроки одной команды могут отвечать с разных устройств
3. Побеждает команда чей любой участник нажал первым

### Интернет-игра с удаленными участниками

1. Разверните сервер на VPS (например, DigitalOcean, Heroku)
2. Получите публичный IP или домен
3. Разошлите ссылки участникам
4. Используйте видео-конференцию для голосового общения

## Советы по проведению

### Для ведущего
- Подготовьте список ответов beforehand
- Проверьте звук заранее
- Имейте запасные вопросы
- Следите за таймингом игры
- Будьте готовы к нестандартным ответам

### Для участников
- Выберите капитана команды
- Обсудите ответы перед нажатием кнопки
- Не кричите одновременно
- Уважайте правила

### Технические проверки
- Проверьте интернет соединение
- Подключите устройства к одной сети (LAN для лучших результатов)
- Проверьте звуковые колонки
- Имейте зарядные устройства для мобильных
- Подготовьте план B (бумажные вопросы на случай сбоя техники)

## FAQ

**Q: Сколько людей может играть?**  
A: От 2 до 20+. Оптимально 6-12 человек, разделенных на 2 команды.

**Q: Можно ли играть без интернета?**  
A: Да, если все устройства в одной локальной сети и не используете внешние ресурсы.

**Q: Как долго длится игра?**  
A: 30-90 минут, зависит от количества раундов и темпа игры.

**Q: Можно ли добавить свои вопросы?**  
A: Да, используйте редактор пакетов или создайте JSON файл вручную.

**Q: Как сохранить результаты игры?**  
A: Сделайте скриншот табло или запишите очки на бумаге.

**Q: Можно ли изменить очки во время игры?**  
A: Да, вручную через кнопку "Начислить очки" в панели ведущего.
