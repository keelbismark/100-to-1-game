# Звуковые эффекты для игры

Эту папку нужно заполнить звуковыми файлами в формате MP3.

## Требуемые файлы:

1. **reveal.mp3** - Звук открытия правильного ответа (Дзинь!)
   - Рекомендации: короткий, звонкий звук
   - Длительность: ~0.5-1 секунда

2. **mistake.mp3** - Звук промаха (Гудок)
   - Рекомендации: низкий звук, похожий на неправильный ответ
   - Длительность: ~0.5 секунды

3. **wrong.mp3** - Звук ошибки (когда 3 промаха)
   - Рекомендации: более длительный и неприятный звук
   - Длительность: ~1-2 секунды

4. **winner.mp3** - Звук победителя (кто первым нажал кнопку)
   - Рекомендации: торжественный звук
   - Длительность: ~1 секунда

## Где найти звуки:

1. Бесплатные звуковые библиотеки:
   - Freesound.org
   - Zapsplat.com
   - Mixkit.co (free sound effects)

2. Или создайте свои с помощью:
   - Audacity (бесплатный аудиоредактор)
   - Online generators

## Формат:

- Формат: MP3
- Качество: 128-192 kbps
- Моно или Stereo (нормально оба варианта)
