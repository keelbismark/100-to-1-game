## Load Testing

### Базовый тест
1. Создайте 3 комнаты
2. Подключите по 2 игрока в каждую
3. Запустите игры параллельно
4. Проверьте что все работает без лагов

### Умеренный тест
1. Создайте 10 комнат
2. Подключите по 2 игрока в каждую
3. Запустите игры параллельно
4. Проверьте нагрузку на сервер

### Тяжелый тест
1. Создайте 50 комнат
2. Подключите по 2 игрока в каждую
3. Запустите игры параллельно
4. Мониторьте производительность

## Автоматизация

Планируется добавить:

### Unit Tests
```javascript
// Пример
describe('GameManager', () => {
  test('должен сохранять состояние игры', () => {
    const gameState = { ... }
    expect(GameManager.saveGameState(gameState)).toBe(true)
  })
})
```

### E2E Tests
```javascript
// Пример
test('создание игры должно работать', async ({ page }) => {
  await page.goto('http://localhost:3000')
  await page.click('button:has-text("Создать игру")')
  await expect(page).toHaveURL(/\/game\/[A-Z0-9]{4}\/admin/)
})
```
